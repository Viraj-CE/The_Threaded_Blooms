# The Threaded Blooms

A premium e-commerce platform for handcrafted natural candles and home fragrances.

## Project Overview

The Threaded Blooms is designed to provide a seamless and aesthetic shopping experience for customers seeking high-quality, natural home fragrance products. The application features a complete product catalog, shopping cart, and mock checkout flow.

## Key Features

- **Storefront**: A beautiful, responsive product listing page with category filtering.
- **Search**: Integrated site-wide search to help users find specific products.
- **Product Details**: Comprehensive product views including fragrance notes and user reviews.
- **Shopping Cart**: A persistent cart system with real-time total calculation.
- **User Dashboard**:
  - **Orders Portal**: Track past orders and view status updates.
  - **Profile Management**: Manage personal details and multiple shipping addresses.
- **Responsive Design**: Fully optimized for mobile and desktop experiences.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Routing**: React Router
- **State Management**: React Context API
- **UI Components**: Shadcn/UI (Radix UI)

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Start the development server: `npm run dev`
4.  Build for production: `npm run build`
